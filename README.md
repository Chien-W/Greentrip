### 部分文件约束
1. src/assets 放置静态资源(jpg,png,svg)
2. src/components 放小型组件
3. src/views 放大组件(我的，首页，发现，活动，商城)

### css约束
&emsp;使用rem(1rem == 20px)

### eslint约束
1. 缩进格数应为2
2. 句尾不应该加 ;

#### 安装依赖
```
npm install
```

#### 本地运行(开发环境)
```
npm run serve
```

#### 打包
```
npm run build
```
